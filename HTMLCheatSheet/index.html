HTML Reference Sheet

TAG NAME DESCRIPTION
<html> html defines an html document
<title> title defines a title for the document
<p> paragraph defines a paragraph
<br> break inserts a single line break
<time> time defines a date/time
<form> form defines an html form for user input
<input> input defines an input control
<textarea> textarea defines a multiline input control
<ul> unordered list defines an unordered list
<ol> ordered list defines an ordered list
<li> list item defines a list item
<th> table header defines a header cell in a table
<table> table defines a table
<tr> table row defines a row in a table
<thead> table header Groups the header content in a table
<tbody> table body groups body content in a table
<tfoot> table foot groups the footer content in a table
<col> column specifies column properties for each column within a colgroup element
<colgroup> column group specifies a group of one or more columns in a table for formatting
